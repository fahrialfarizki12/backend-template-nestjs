# 🔐 Backend Auth Template

Template backend authentication ini dibuat menggunakan **NestJS** (atau sesuaikan dengan stack kamu) untuk kebutuhan sistem login, register, OTP verification, JWT authentication, dan integrasi dengan frontend (Next.js / React / Vue).

Project ini dirancang sebagai **starter backend reusable** untuk berbagai aplikasi seperti dashboard, SaaS, atau aplikasi fullstack.

---

## 🚀 Tech Stack

- Node.js
- NestJS
- TypeScript
- JWT (JSON Web Token)
- Passport (optional)
- Prisma / Sequelize (sesuaikan)
- Nodemailer (OTP Email Service)
- bcrypt (password hashing)

---

## ✨ Fitur Utama

- ✅ Register user
- ✅ Login user
- ✅ JWT Authentication
- ✅ Role-based access (admin/user)
- ✅ Email OTP verification
- ✅ Resend OTP
- ✅ Forgot password
- ✅ Reset password
- ✅ Cookie-based auth (optional)
- ✅ Middleware protection route

## 📦 Struktur Project

# ⚡ Next.js Auth Template Frontend

Frontend ini dibuat menggunakan **Next.js** untuk terhubung dengan backend authentication (NestJS / Express API). Project ini mendukung login, register, OTP verification, dan protected route menggunakan JWT.

---

## 🚀 Tech Stack

- Next.js (App Router / Pages Router)
- TypeScript
- Axios
- JWT Auth
- Cookie / LocalStorage
- Tailwind CSS (optional)

---

## ✨ Fitur

- ✅ Register user
- ✅ Login user
- ✅ OTP verification page
- ✅ Forgot password
- ✅ Reset password
- ✅ Protected routes
- ✅ Role-based UI (user/admin)
- ✅ Auto redirect jika belum login
- ✅ Token handling (cookie/localStorage)
